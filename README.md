# TikDown

Il y a deux façaon de telecharger les videos :

1- Sur le cmd
![Command Line](image/cmd.png)

2-Sur chrome
![Web Interface](image/web.png)

Sur un cmd : il suffit d'éxecuter python down.py puis coller le lien du video TikTok .  
  
Sur chrome :  
a- Installer l'extension dans le dossier "extension".  
b- Executer :  
cd server  
python server.py  
  
c- Ouvrir la video et cliquer sur le bouton "Telecharger" .  

## Le chocolat est finit .
